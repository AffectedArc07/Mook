# Mook

Mokou fortune picker Discord bot.

## Building

Should be as simple as running `mvn package` then getting your jar from the `out` folder.

## Running

1. Copy `config.toml.example` to `config.toml`
2. Fill out all configuration sections
3. Launch the bot
